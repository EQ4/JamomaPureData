Welcome Jamoma for Puredata page

You will find both nightly builds and releases there. We are trying to provides binary for Linux 32 & 64 bit, Raspberry Pi, OS X and Windows (as soon as possible).

To use it, download the archive corresponding to your platform and unpack it in your ~/pd-externals folder on Linux, ~/Library/Pd folder on Mac OS X and somewhere on Windows.
